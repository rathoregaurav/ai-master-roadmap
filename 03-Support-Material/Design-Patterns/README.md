# AI Design Patterns

Reusable patterns for AI systems.

## Core Patterns

- **Router** - Route input to appropriate handler
- **Reflection** - Generate → Critique → Refine
- **Supervisor** - Orchestrate multiple workers
- **Planner-Executor** - Plan first, then execute
- **Human-in-the-Loop (HITL)** - Require approval for sensitive actions
- **Debate** - Multiple agents debate for better answers
- **Swarm** - Many simple agents working together
