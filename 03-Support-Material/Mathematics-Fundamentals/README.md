# Mathematics Fundamentals for AI Engineering

Conceptual math understanding - no PhD required!

## Key Topics

- Vectors & Vector Operations
- Cosine Similarity (most important!)
- Matrix Operations (intuition only)
- Conditional Probability & Bayes' Theorem
- Precision, Recall, F1, ROC/AUC

## What You Need to Know

You don't need to derive proofs. You need to:

1. **Understand what cosine similarity measures**
2. **Know when to use precision vs recall**
3. **Have intuition about how vectors represent meaning**
