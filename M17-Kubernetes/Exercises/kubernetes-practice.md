# M17 Practice: Kubernetes

## Beginner Exercises

1. Read `ai-api-deployment.yaml`.
2. Identify replicas, image, container port, and health check.
3. Read `ai-api-service.yaml`.
4. Explain how the service finds the pods.
5. Run `manifest_summary.py`.

## Advanced Exercises

1. Add environment variables for vector database URL.
2. Add a separate worker deployment.
3. Add an HPA design note.
4. Add GPU scheduling notes for a model server.
5. Write a rollback plan for a failed deployment.

## Done When

- [ ] You can explain pod, deployment, service, and ingress.
- [ ] You can read basic YAML manifests.
- [ ] You can explain why readiness probes matter.
- [ ] You can describe when Kubernetes is useful for AI.
