# M4 Practice: Prompt Engineering

## Beginner Exercises

1. Write a system prompt for a resume feedback assistant.
2. Convert a free-form prompt into a JSON-output prompt.
3. Add XML delimiters around user-provided content.
4. Create three few-shot examples for classification.

## Advanced Exercises

1. Build a prompt regression table with 10 test inputs.
2. Add refusal behavior for unsafe or unsupported requests.
3. Compare zero-shot vs few-shot performance.
4. Add a Pydantic model that validates the output schema.

## Done When

- [ ] You can explain prompt as contract.
- [ ] You can separate instruction from data.
- [ ] You can validate structured output.
- [ ] You can test prompts after changes.
