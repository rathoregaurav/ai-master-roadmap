# Environment Checklist Exercise

## Goal

Create a clean local AI engineering environment.

## Tasks

- [ ] Install or confirm Python 3.11+.
- [ ] Create a virtual environment.
- [ ] Install base packages from `Project-Boilerplate/requirements.txt`.
- [ ] Create `.env.example`.
- [ ] Run `python app.py`.
- [ ] Explain each setup step in your notes.

## Stretch Tasks

- [ ] Add Dockerfile.
- [ ] Build a container image.
- [ ] Run the container locally.
- [ ] Add a health check command.

## Reflection

Write answers:

1. What broke during setup?
2. How did you debug it?
3. What would make this setup easier for a teammate?
