# M3 Practice: LLM Fundamentals

## Beginner Exercises

1. Explain tokens using a paragraph from your own notes.
2. Write three prompts with different temperatures and compare behavior.
3. Estimate why long prompts cost more.
4. Create a timeout failure intentionally with a fake endpoint.

## Advanced Exercises

1. Wrap the LLM client in a retry function.
2. Add cost estimation fields to the response.
3. Add a model router that chooses a small or large model based on task.
4. Add structured output validation with Pydantic.
