# M15 Practice: AI System Design

## Beginner Exercises

1. Read the RAG + Agent design template.
2. Draw your current RAG architecture.
3. Mark where caching belongs.
4. Mark what should run in a background queue.
5. Add one fallback path.

## Advanced Exercises

1. Add tenant-aware retrieval filters.
2. Add a circuit breaker around model calls.
3. Add semantic cache design.
4. Add rate limits.
5. Write a one-page system design for the Phase 4 deliverable.

## Done When

- [ ] You can explain caching vs queues.
- [ ] You can explain circuit breakers.
- [ ] You can draw a production RAG architecture.
- [ ] You can discuss trade-offs in an interview.
