# M21 Practice: Enterprise AI

## Beginner Exercises

1. Run `registry_and_audit.py`.
2. Add a second prompt version.
3. Add a second model record.
4. Record a tool-call audit event.
5. Create a lineage record for a document chunk.

1. Add tenant ID to audit logs.
2. Add approval status to model records.
3. Add prompt rollback metadata.
4. Add policy checks before model selection.
5. Design an admin panel table for prompts and models.
