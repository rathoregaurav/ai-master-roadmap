## ⚠️ Watch Out For These

### Common Pitfalls

1. **Trying to learn everything about ML first**
   - You don't need to train models from scratch
   - Focus on using APIs first

2. **Over-engineering from day 1**
   - Start simple, iterate
   - Your backend instincts might make you overcomplicate

3. **Ignoring evaluation**
   - AI systems need different testing approaches
   - Build evaluation early
