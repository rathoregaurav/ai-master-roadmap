# M13 Practice: AI Security

## Beginner Exercises

1. Run `pii_and_injection_guard.py`.
2. Add one new prompt injection phrase.
3. Add a test input with an email and phone number.
4. Explain why detection by phrase matching is incomplete.

## Advanced Exercises

1. Add risk labels to tools from Phase 3.
2. Add approval checks for write actions.
3. Add output scanning for leaked email addresses.
4. Create 10 security test cases.
5. Write a short threat model for your RAG app.

## Done When

- [ ] You can explain direct and indirect prompt injection.
- [ ] You can scrub simple PII.

- [ ] You can write a basic AI threat model.
