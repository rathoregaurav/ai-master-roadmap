# Open Source & Local Models

Running models locally and on-prem.

## Model Comparison

| Model | Params | Quantized RAM | Speed | Best For |
|-------|--------|--------------|-------|----------|
| Llama 3.2 3B | 3B | 2GB | 50-80 tok/s | Classification |
| Mistral 7B | 7B | 4GB | 30-50 tok/s | General purpose |
| Qwen 2.5 7B | 7B | 4GB | 30-50 tok/s | Code, math |
| Llama 3.1 8B | 8B | 4.5GB | 25-40 tok/s | Chat |

## Tools

- **Ollama** - Easiest way to run models locally
- **vLLM** - High-throughput inference
- **TGI** - Text Generation Inference
- **GGUF** - Quantization format
