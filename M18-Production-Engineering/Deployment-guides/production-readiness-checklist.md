# Production Readiness Checklist

## Code

- [ ] Code is formatted.
- [ ] Tests pass.
- [ ] No secrets in code.
- [ ] Dependencies are documented.

## AI Quality

- [ ] Golden dataset exists.
- [ ] Prompt regression tests pass.
- [ ] Retrieval metrics are recorded.
- [ ] Tool-call tests pass.

## Security

- [ ] Input validation exists.
- [ ] PII handling is documented.
- [ ] Prompt injection tests exist.
- [ ] Tool permissions are defined.

## Observability

- [ ] Request IDs are logged.
- [ ] Latency is measured.
- [ ] Model name is tracked.
- [ ] Token/cost estimate is tracked.
- [ ] Errors are visible.

## Deployment

- [ ] Environment variables are documented.
- [ ] Health endpoint exists.
- [ ] Rollback plan exists.
- [ ] Load test result is recorded.
