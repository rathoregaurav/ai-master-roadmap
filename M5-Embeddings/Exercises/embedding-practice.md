# M5 Practice: Embeddings

## Beginner Exercises

1. Explain cosine similarity in your own words.
2. Add two documents to `simple_semantic_search.py`.
3. Try three different queries and record results.
4. Explain where bag-of-words fails compared with real embeddings.

## Advanced Exercises

1. Add IDF-style weighting.
2. Normalize text by removing punctuation.
3. Add Recall@K evaluation for five query/document pairs.
4. Compare keyword and vector-style scoring.

## Done When

- [ ] You can implement cosine similarity from memory.
- [ ] You can explain semantic search to a beginner.
- [ ] You can name two failure modes of embeddings.
- [ ] You can design a small embedding benchmark.
