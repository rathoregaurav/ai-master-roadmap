# M9 Practice: MCP

## Beginner Exercises

1. Run `mcp_style_server.py`.
2. Add one new resource for prompt engineering.
3. Add one new read-only tool.
4. Explain the difference between a tool and a resource.

## Advanced Exercises

1. Add permission labels to tools.
2. Add audit logs.
3. Add approval requirement for write tools.
4. Convert the example into a real MCP server later.

## Done When

- [ ] You can explain MCP in plain English.
- [ ] You can separate tools from resources.
- [ ] You can design a safe internal connector.
- [ ] You can explain why enterprises care about MCP.
