# M2 Practice: Backend Engineering for AI

## Beginner Exercises

1. Run `fastapi_sse_example.py` with `uvicorn`.
2. Open `/health`.
3. Call `/stream?prompt=hello`.
4. Add a POST endpoint that accepts a Pydantic body.

## Advanced Exercises

1. Add auth as a FastAPI dependency.
2. Add request IDs to logs.
3. Add a background task that stores prompt history.
4. Add rate-limit notes to the README.
5. Add OpenAPI examples for each route.
