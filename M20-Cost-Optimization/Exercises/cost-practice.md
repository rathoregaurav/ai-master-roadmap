# M20 Practice: Cost Optimization

## Beginner Exercises

1. Run `model_router.py`.
2. Add a task type for translation.
3. Change token counts and observe estimated cost.
4. Write three ways to reduce cost in your RAG project.

## Advanced Exercises

1. Add per-model pricing.
2. Add a semantic cache stub.
3. Add a rule that blocks very large prompts.
4. Add quality evaluation before accepting cheaper model routing.
5. Create a cost report for 100 sample requests.

## Done When

- [ ] You can estimate basic model cost.
- [ ] You can explain model routing.
- [ ] You can explain semantic caching.
- [ ] You can balance cost and quality with evals.
