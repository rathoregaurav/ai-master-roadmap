# M8 Practice: Tool Calling

## Beginner Exercises

1. Run `tool_registry.py`.
2. Add a `subtract` operation.
3. Call an unknown tool and inspect the error.
4. Call a tool with missing arguments.

## Advanced Exercises

1. Add JSON schema definitions for each tool.
2. Add permission levels.
3. Add a fake weather API tool.
4. Add parallel calls for safe read-only tools.
5. Add tool-call evaluation cases.

## Done When

- [ ] You can explain tool schema.
- [ ] You can validate tool arguments.
- [ ] You can handle tool errors.
- [ ] You can explain why tool permissions matter.
