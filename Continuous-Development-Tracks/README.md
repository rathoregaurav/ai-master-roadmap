# Continuous Development Tracks

Four tracks to run in parallel with your main learning.

## Track A: Portfolio Polishing

- Update READMEs
- Add architecture diagrams
- Create demo videos
- Refine project documentation

## Track B: Interview Prep

- 20 questions/week
- System design practice
- Whiteboarding exercises
- Behavioral prep

## Track C: AI News

- ArXiv paper digest (30 min/week)
- HuggingFace updates
- Industry news aggregation

## Track D: Engineering Hygiene

- Testing in every project
- Logging & Metrics
- Security best practices
- Code reviews

## Daily Schedule Template

```
8:00-8:30   | ArXiv/HuggingFace digest
12:00-12:30 | 3 interview questions
15:00-15:30 | Engineering hygiene (tests)
Weekend     | Portfolio polish (2-3 hrs)
```
