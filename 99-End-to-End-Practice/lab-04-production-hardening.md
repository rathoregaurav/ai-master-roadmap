# Lab 4: Production Hardening

## Goal

Take a simple AI workflow and add Phase 4 engineering controls.

## Beginner Version

Add:

- request ID
- latency tracking
- prompt injection phrase detection
- PII scrubbing
- simple model routing
- cost estimate

## Advanced Version

Add:

- trace spans
- cache hit/miss tracking
- circuit breaker
- output validation
- security test cases
- benchmark report

## Deliverable

A CLI or API that shows:

- safe input text
- security flags
- selected model
- estimated cost
- latency
- final response

## Reflection Questions

1. What did observability reveal?
2. What security checks are too weak?
3. What should be cached?
4. What model routing rule saved cost?
5. What would you monitor in production?
