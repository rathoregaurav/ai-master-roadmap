# M12 Practice: AI Observability

## Beginner Exercises

1. Run `request_trace.py`.
2. Add a `guardrail` span.
3. Add total request latency.
4. Add `model_name` to model metadata.

## Advanced Exercises

1. Save traces as JSON lines.
2. Add cost estimation.
3. Add cache hit/miss tracking.
4. Add retrieved source file names.
5. Create a simple dashboard summary from trace files.

## Done When

- [ ] You can explain a trace.
- [ ] You can identify slow workflow steps.
- [ ] You can track token and cost metadata.
- [ ] You can avoid logging sensitive raw user data.
