# M6 Practice: RAG

## Beginner Exercises

1. Chunk a plain text document.
2. Store chunks in an in-memory list.
3. Search chunks with simple similarity.
4. Return answer context with chunk IDs.
5. Create five golden questions.

## Advanced Exercises

1. Add metadata filters.
2. Add hybrid search scoring.
3. Add query rewriting.
4. Add reranking.
5. Calculate Recall@K and MRR.
6. Build a simple evaluation report.

## Done When

- [ ] You can trace a RAG answer back to source chunks.
- [ ] You can explain why retrieval failed.
- [ ] You can improve retrieval and prove the metric changed.
- [ ] You can describe when RAG is better than fine-tuning.
