# AI Chat UX Patterns

## Essential States

- empty state
- user composing
- request submitted
- streaming response
- tool running
- citation loading
- complete answer
- error
- feedback submitted

## Trust Patterns

- show citations
- show retrieved source names
- separate answer from evidence
- say when answer is uncertain
- allow correction
- preserve conversation history

## Feedback Data

Store:

- response ID
- user rating
- optional comment
- cited sources
- model version
- prompt version
- latency
- tenant or workspace ID
