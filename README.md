# AI Master Roadmap

Complete learning path for becoming an enterprise-ready AI engineer. This repository is designed for four outcomes at the same time:

- Learn modern AI engineering from first principles.
- Build portfolio projects that prove skill, not only knowledge.
- Prepare for technical and system design interviews.
- Practice real production work: evaluation, security, observability, cost, deployment, and governance.

## How To Use This Repository

For each module:

1. Read the module `README.md`.
2. Study the matching section in [COMPREHENSIVE-THEORY-GUIDE.md](./COMPREHENSIVE-THEORY-GUIDE.md).
3. Run at least one code example.
4. Complete the exercise.
5. Answer the interview questions out loud.
6. Add the topic to a portfolio project or lab.
7. Check the module against [ENTERPRISE-READINESS-MATRIX.md](./ENTERPRISE-READINESS-MATRIX.md).

## Definition Of Done

A module is not complete until you can show:

- Notes written in your own words.
- Working code or a runnable demo.
- One architecture diagram or sequence flow.
- Tests, eval cases, or manual verification notes.
- Interview answers for the core questions.
- Production notes covering security, observability, cost, and failure modes.
